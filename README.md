# Sales Data Analysis Project

## Overview
This project performs an exploratory data analysis (EDA) on a sample sales dataset. It analyzes revenue trends, product category performance, and geographic distribution, visualizing the results in a cohesive dashboard.

## Dashboard Output
![Sales Dashboard](sales_dashboard.png)

## Educational Purpose
> [!IMPORTANT]
> **This project was created primarily for educational purposes.**
> It is designed to demonstrate:
> *   Data manipulation using `pandas`
> *   Data visualization using `matplotlib`
> *   Basic data science workflow in Python

## Setup & Running

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Download Data**:
    Ensure `sales_data.csv` is present in the directory.
3.  **Run the Dashboard**:
    Start the interactive Streamlit application:
    ```bash
    streamlit run app.py
    ```
    This will open the dashboard in your default web browser (usually at `http://localhost:8501`).
